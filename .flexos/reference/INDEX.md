<flex_block type="instructions">
Reference docs are SDK/framework documentation for the builder AI.
Imported or generated based on the project's technical specs.

Each file covers one technology: key APIs, patterns, gotchas, and examples.
The build AI reads these to know how to use the chosen frameworks correctly.

NAMING: NNN-{slug}.md
EXAMPLE: 001-nuxt4.md, 002-convex.md, 003-twilio.md

Generate reference docs for every technology in the technical stack spec.
</flex_block>
