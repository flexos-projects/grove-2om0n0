# Welcome to Grove

This workspace is organised as a collection of spec files.
Each spec type has its own folder.

## How to Work

1. **Talk to Flexi** — describe what you want to build or change
2. **Specs update** — Flexi creates and updates spec files in the right folders
3. **Core docs regenerate** — holistic summaries in docs/ stay current
4. **Prototypes generate** — visual HTML previews of your pages
5. **Build executes** — specs + prototypes → production Nuxt 4 app

## Spec Types

- **Vision** (`vision/`) — goals, constraints, audience, brand identity
- **Features** (`features/`) — what the app does
- **Pages** (`pages/`) — routes, components, layouts
- **Flows** (`flows/`) — user journeys and system processes
- **Database** (`database/`) — data model and APIs
- **Design** (`design/`) — tokens, patterns, visual system
- **Technical** (`technical/`) — stack, integrations, infrastructure
- **Content** (`content/`) — copy, seed data, CMS records
